SELF-SITE
So simple, it's complicated - ...or is it?
WHY?
  • Individual learning curve
  • No dependency, not even Internet connection*
  • Full self-containment*
  • Fully modifiable/adjustable
  • Modular, up- and down-scalable
  • Capable of almost anything
    * unless dependencies/resource links intentionally invoked
 
HOW?
  • Add '?admin' to the URL
  • Type into panel for text edits - that's all. That's all needed. But...

  • Editor: double-click panel and edit HTML/CSS (-eek!-)
    • WAIT! Use snippets: create templates with 'self-template' tool and replace or add in source code view (There: de-eek-ed!)
    • Click somewhere off panel, to turn off editor

  • Messed up? [CTRL]+[Z] = Undo
  • When done, click [SAVE] on bottom right (visible after changes), and OVERWRITE original.
  • If in doubt or if backup needed, save under different name and swap names with original.
  • If new page created, save under new page name.
 
AND THEN?
  • Find someone with HTML/CSS/JS skills.
  • Be someone with HTML/CSS/JS skills.
  • Ignore all geek-stuff and use as-is/as-you-can. You will get there automatically, no need to fuzz.
  • Keep on local computer, own phone, some network, or on the Internet - as needed.
